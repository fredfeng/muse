\headerbox{Objective and Goals}{name=objective,column=0,row=0}{
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
{\sc Hunter} is a next generation code reuse tool that finds, adapts and synthesizes executable codes to
\begin{enumerate}
\item {\bf \emph{Increase productivity:}} it allows programmers focus more on creative tasks.
\item {\bf \emph{Decrease buggy code:}} it could reuse code from well-tested corpus and reduce the likelihood of buggy implementation.
\end{enumerate}
}
